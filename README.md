Grape in Rails
====

Use grape with swagger in rails.

## Installation

See demo for start:

Clone the repository.

```sh
$ git clone https://github.com/ajream/grape-rails.git
```

Change directory

```sh
$ cd grape-rails
```

Run bundler

```sh
$ bundle install
```

Preparing database

```sh
$ bin/rails db:migrate
```

Start the Rails server

```sh
$ bin/rails s
```

Open your browser, and visit `http://localhost:3000`
